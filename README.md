Yocto
=====

A Minimal JSON Parser & Printer for Haskell

[![Build Status](https://travis-ci.org/ajg/yocto.png?branch=master)](https://travis-ci.org/ajg/yocto)

Dependencies
------------

The only requirements are the `parsec` and `containers` packages.
